# Distance Metrics in KNN
### Why Euclidean Isn't Always the Answer

A machine learning tutorial comparing Euclidean, Manhattan, and Minkowski distance metrics in K-Nearest Neighbours classification, using distance contour visualisations on the Wine Quality dataset.

## Learning Objectives
1. Explain how KNN uses distance to classify new points
2. Compare the geometry of Euclidean, Manhattan, and Minkowski distances
3. Demonstrate how distance metric choice changes classification outcomes
4. Understand why feature scaling is critical for distance-based methods
